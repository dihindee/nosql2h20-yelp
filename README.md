# nosql2h20-yelp


## Local deploy
Приложение работает на 8080 порту, mongodb 27017

###backend
Сервер приложения server.js
```
node backend/server.js
```

###mongo

Запуск mongodb
```
sudo systemctl start mongod
```

Остановка mongodb

```
sudo systemctl stop mongod
```
